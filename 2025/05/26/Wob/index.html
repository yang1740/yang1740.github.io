Web 学习笔记

📅 日期：2025-05-26
✍️ 作者：杨鸿源
🔖 标签：Web、前端、学习笔记

1️⃣ Web 是什么？

Web 是一种新兴的 Web 技术栈，结合了 WebAssembly、WebGPU 和浏览器新能力，旨在实现更高性能的前端应用。Wob 不是一个框架，而是一种技术方向和生态组合。

  • W: WebAssembly（WASM）
  • e: OffscreenCanvas（离屏渲染）
  • B: WebGPU（浏览器 GPU 加速）

2️⃣ 核心概念

WebAssembly

  • 一种高性能二进制格式，可以让非 JavaScript 语言编译为浏览器可运行的代码。
  • 优势：接近原生性能，适合高性能应用。

WebGPU

  • 浏览器的 GPU 接口，替代 WebGL。
  • 优势：底层 GPU 操作，现代图形 API，性能更强。

OffscreenCanvas

  • 允许在主线程以外的 Worker 线程中进行画布绘制，避免主线程阻塞。
  • 适合动画、图形处理等。

3️⃣ Web 应用场景

✅ 图形学 / 游戏开发
✅ 高性能数据可视化
✅ AI / ML 前端推理
✅ CAD / 3D 建模
✅ 视频后处理 / 滤镜

4️⃣ 学习资源推荐

  • WebAssembly 官网
  • WebGPU 官方文档
  • MDN WebGPU 指南
  • OffscreenCanvas 介绍

5️⃣ 我的思考与总结

  • Wob 结合了前端新技术的力量，为复杂应用打开了新大门。
  • 学习 Web 的过程中，需要掌握：
    • 编译工具链（如 Rust、C/C++ -> WASM）
    • 图形 API 基础（如 WebGPU / OpenGL）
    • 前端与 Worker 线程协作
